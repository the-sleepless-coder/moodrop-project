# 지역별 실시간 날씨 정보 제공 웹 서비스

## 프로젝트 개요

**공공데이터 포털의 기상청 단기예보 API**를 활용하여, 웹 페이지에서 사용자가 원하는 지역을 검색하면 해당 지역의 실시간 날씨 정보를 제공하는 서비스입니다.  
지역명(예: "서울특별시 종로구 청운효자동") 입력 시 기온, 습도, 하늘 상태 등 상세 예보를 바로 확인할 수 있습니다.

---

## 주요 기능

- **지역 검색 입력**: 웹에서 행정구역명(시/도, 구/군, 동/읍/면 등)을 직접 입력
- **지점 변환**: 입력 지역명을 격자(X, Y) 좌표로 변환 (엑셀 데이터 활용)
- **기상청 예보 호출**: 행정구역의 격자 좌표에 맞춰 기상청 초단기예보 API 호출
- **날씨 정보 출력**:
  - 기온
  - 습도
  - 하늘 상태 (맑음, 구름 많음, 흐림, 비, 눈 등)

---

## 사용 기술

- Python (Flask)
- Pandas (지역명→격자좌표 변환)
- requests, xmltodict (API 통신 및 데이터 파싱)
- HTML (index.html 템플릿)
- 공공데이터포털 기상청 단기예보 API

## 실행 및 사용 방법

1. **패키지 설치**
pip install flask pandas requests xmltodict

2. **프로그램 실행**
python app.py
- 기본 포트: 5000

3. **웹 접속**  
브라우저에서 `http://localhost:5000` 접속

4. **지역명 입력**
- *예:* `서울특별시 종로구 청운효자동`
- 입력 후 [검색/확인] 버튼 클릭

5. **결과 예시**
서울특별시 종로구 청운효자동
날씨 : 맑음
온도 : 18.2ºC
습도 : 43%

---

## 주요 구현 코드

- **지역→격자 변환**: `find_grid_position()` 함수에서 엑셀 파일 활용
- **공공데이터 호출**: `forecast()` 함수에서 기상청 예보를 격자좌표별로 수신
- **결과 가공/출력**: `proc_weather()` 함수에서 하늘상태, 온도, 습도 등 가공해 출력
